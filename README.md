
# Consumer Behavior Analysis

## Project Overview
This project analyzes consumer shopping behavior to identify purchasing trends, customer segments, and business insights that help improve marketing strategies and customer retention.

## Business Problem
A leading retail company wants to better understand customer shopping behavior to improve sales and customer satisfaction.

## Tools Used
- Python
- Pandas
- NumPy
- SQL
- Power BI
- GitHub

## Project Workflow
- Data Cleaning
- Exploratory Data Analysis
- SQL Business Queries
- Dashboard Development
- Business Insights

## Key Insights
- Top selling product categories
- Customer Segmentation
- Seasonal Sales Trends
- Preferred Payment Methods
- Discount Impact
- Repeat Purchase Analysis
